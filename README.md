# Order System Frontend

A Vue 3 frontend for the Order System application built with Vite.

## Overview

This repository contains the frontend application for an order management system.
It provides a login page and a dashboard page, with client-side routing and API communication via Axios.

## Features

- Vue 3 single-page application
- Vue Router navigation
- Axios HTTP client for backend API calls
- Login and dashboard views
- Vite-powered development and build workflow

## Technology stack

- Vue 3
- Vite
- Vue Router
- Axios

## Requirements

- Node.js 18+ (recommended)
- npm

## Setup

Install dependencies:

```bash

npm install
npm install axios vue-router
npm run dev
```

## Development

Run the development server:

```bash
npm run dev
```

Then open the local URL shown in the terminal, usually `http://localhost:5173`.
